# Sphere Council Wallet

A sovereign multi-chain wallet—"Mask of the Meta"—building on MetaMask but extended with cross-chain support, identity, security, and AI features. This repository contains the initial scaffolding for the project.

## Project Structure

- `src/` – source for the browser extension prototype
- `public/` – static assets and extension manifest
- `docs/` – design notes and planning documents

## Getting Started

The project is in an early experimental phase. After cloning:

```bash
npm install
npm run build
```

Both commands are placeholders; no production build exists yet.
